# Nettoyeur de Noms de Fichiers

Application web progressive (PWA) permettant de nettoyer les noms de fichiers en supprimant les caractères invalides et en appliquant diverses transformations.  
Deux modes : saisie texte ou upload de fichiers réels.

## Fonctionnalités
- Mode Texte : coller une liste de noms, nettoyer, exporter.
- Mode Upload : glisser-déposer des fichiers, renommage batch, téléchargement individuel ou groupé.
- Options de nettoyage : espaces, casse, préfixe, suffixe, gestion des doublons.
- Personnalisation des caractères interdits.
- Thème clair/sombre, bouton retour en haut.
- Responsive, SEO-friendly.

## Technologies
- HTML5 / CSS3 / JavaScript ES6
- localStorage pour la persistance
- API File pour le mode upload
- Pas de dépendances serveur (100% client)

## Installation
1. Cloner le dépôt.
2. Ouvrir `index.html` dans un navigateur moderne.
3. (Optionnel) Héberger sur Netlify ou autre service statique.

## Utilisation
- **index.html** : mode texte.
- **upload.html** : mode upload avec gestion de fichiers.
